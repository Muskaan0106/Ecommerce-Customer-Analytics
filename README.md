# Customer Purchase & Behavior Analysis

## Objective
Analyze e-commerce customer data to understand purchasing behavior, engagement metrics, and churn patterns.

## Dataset
E-commerce Customer Churn Dataset containing customer demographics, purchases, engagement, and lifetime value.

## Key Analysis Performed
- Customer purchase behavior analysis
- Engagement vs purchase trends
- Lifetime value analysis
- Churn comparison

## Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn

## Key Insights
- High engagement leads to higher customer lifetime value
- Long-term customers contribute significantly to revenue
- Churned customers show lower purchasing behavior
